== System Overview

This architecture consists of two complementary agent patterns working together:

. *Headless Automator* (LangGraph) - Autonomous research agent for open-ended tasks
. *Digital Employee* (Rasa CALM) - Structured conversational agent for high-stakes interactions

Both agents share a common tool layer through MCP (Model Context Protocol).
